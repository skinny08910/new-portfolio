Gerson's Simple Food

Project Info:

Gerson's Simple Food is a restaurant website with a simple yet modern design to it, it's made with Next.js to output the HTML/CSS file and I add some GSAP to animate the mobile menu.

Project Details:
  • Technologies:HTML, CSS, SASS, JavaScript, GSAP, Next.js
